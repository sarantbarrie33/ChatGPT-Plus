# ChatGPT Plus 如何取消自动扣费/订阅

ChatGPT Plus 是一种订阅服务，每个月的订阅费用是 20 美元，每个月都会进行扣费。如果您订阅了 ChatGPT Plus，而帐户余额不足 20 美元，OpenAI 在多次扣费失败后，可能会封禁您的帐户。因此，如果不继续使用 ChatGPT Plus，请及时取消订阅。

## 取消订阅的步骤

### 1. 登录 ChatGPT，点击左下角帐户的头像

![image](https://github.com/sarantbarrie33/ChatGPT-Plus/assets/169967439/308a14e9-ec07-4592-87d5-c9074d9f3094)


### 2. 在弹出的弹窗中，点击「My plan」

![image](https://github.com/sarantbarrie33/ChatGPT-Plus/assets/169967439/d3bcaa71-7888-4382-ad40-635bb8d34850)


### 3. 在弹出的「My plan」弹窗中，选择「Manage my subscription」

![image](https://github.com/sarantbarrie33/ChatGPT-Plus/assets/169967439/9e2f7c08-e053-4d25-9bb2-5e65cd46900e)


### 4. 在跳转的订阅页面，选择「取消方案」/ Cancel

![image](https://github.com/sarantbarrie33/ChatGPT-Plus/assets/169967439/45777550-749c-4935-97fc-cda29a2b15e9)
